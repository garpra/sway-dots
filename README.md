# Sway Dotfiles

> Clean & minimal Sway setup with Arch Linux.

## 🔧 Stack

- **Operating System:** Arch Linux
- **Window Manager:** Sway
- **Theme:** Tokyo Night
- **Terminal:** Foot
- **Shell:** Fish

## 📝 Features

- Minimal and clean Sway configuration
- Tokyo Night color scheme
- Custom keybindings and workspace setup
- Fully automated setup script

## ⚡ Installation

Clone this repo to `$HOME`:

```bash
git clone https://github.com/garpra/dots-sway ~/.dotfiles/dots-sway

cd ~/.dotfiles/dots-sway
```

Run automatic setup:

```bash
./setup.sh
```
